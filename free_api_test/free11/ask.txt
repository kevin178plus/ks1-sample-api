## 公众号定位
- **名称**：《古美西路+》
- **视角**：古美西路各年龄层居民（从学龄儿童到银发老人）的社区生活视角
- **核心领域**：民生服务、教育成长、银发关怀、社区治理、邻里关系
- **调性**：温暖、务实、有烟火气、关注"附近"的生活

## 选题筛选标准（符合任一即可）

### 【高优先级】直接相关
1. **教育政策变动** - 中考分流、师范生培养、小班化教学、课后服务、家校关系
2. **养老/银发政策** - 养老金调整、延迟退休、社区养老、医疗服务、适老化改造
3. **住房/物业政策** - 公积金使用、现房销售、物业费、老旧小区改造、租房政策
4. **社区治理创新** - 基层治理、便民服务、数字化政务、社区活动
5. **民生消费** - 物价波动（菜价、油价、水电）、消费维权、本地商业

### 【中优先级】可本地化解读
- 职场权益（加班文化、劳动保护）→ 关联社区就业人群
- 健康养生（睡眠、饮食、运动）→ 关联全年龄段居民
- 家庭关系（育儿、代际沟通）→ 关联多代同堂家庭
- 文化消费（影视、阅读、休闲）→ 关联社区文化生活

### 【低优先级/排除】
- 纯国际时政（除非影响本地生活，如油价上涨）
- 纯娱乐八卦（除非有社会教育意义）
- 纯科技前沿（除非与日常生活强相关）
- 亚文化、游戏、小众圈层话题
