# 土味情话生成

## 任务配置
- 目标人群: 青春校园 (18-25岁)
- 关系阶段: 初识阶段
- 纪念日: 初次见面纪念日
- 表达角度: 日常甜蜜

## 生成要求
请生成3-5条土味情话/浪漫情话。

【目标人群】
- 年龄段: 18-25岁 (青春校园)
- 风格要求: 清新、纯真、略带青涩，可以用校园元素、游戏梗、网络用语

【关系阶段】
- 阶段: 初识阶段 (1-30天)
- 背景: 刚认识不久，还在了解对方，略带试探和好奇

【纪念日场景】
- 纪念日: 初次见面纪念日
- 情感基调: 命中注定、一见钟情、感谢相遇
- 相关元素: 第一次见面, 初遇, 相遇

【表达角度】
- 角度: 日常甜蜜
- 描述: 日常生活中的小甜蜜、小确幸
- 参考: 早安晚安, 今天吃什么, 下班接你

【输出要求】
1. 生成3-5条情话，每条风格可以略有不同
2. 情话要符合上述所有维度的要求
3. 可以有一些是土味情话（带点幽默或套路），也可以有一些是真挚动人的
4. 每条情话后面简要说明适合什么场景使用
5. 使用第一人称"我"对"你"的口吻

【格式示例】
1. "[情话内容]"
   💡 适合场景: [说明]

2. "[情话内容]"  
   💡 适合场景: [说明]
