## 二、Python 版版本详解

### 2.1 独立服务（需要单独启动）

| 服务 | 文件路径 | 端口 | 说明 |
|------|----------|------|------|
| Free8 服务 | `free_api_test/free8/friendli_service.py` | 5008 | Friendli 独立服务 |

> **注意**：Free5 服务已停用（iFlow SDK 已废弃）

### 2.2 主代理服务

| 版本 | 文件路径 | 端口 | 上游数量 | 特点 |
|------|----------|------|----------|------|
| **v3 原版** | `multi_free_api_proxy/multi_free_api_proxy_v3.py` | 5000 | 最多20个 | 功能完整，支持权重管理、故障转移 |
| **v3 优化版（推荐）** | `multi_free_api_proxy/multi_free_api_proxy_v3_optimized.py` | 5000 | 最多20个 | 模块化设计(app_state.py/config.py/errors.py)，**推荐使用** |

> **历史版本**：`local_api_proxy.py` (v1 单文件版) 已移除，不再维护

### 2.3 版本对比

| 功能 | v3 原版 | v3 优化版 |
|------|---------|-----------|
| 多上游支持 | ✅ | ✅ |
| 权重管理 | ✅ | ✅ |
| 故障转移 | ✅ | ✅ |
| free8 独立服务路由 | ✅ | ✅ |
| 并发控制 | ✅ | ✅ |
| 文件监控自动重启 | ✅ | ✅ |
| Debug 页面 | ✅ | ✅ |
| 代码模块化 | ❌ | ✅ |
| 模块文件 | - | `config.py`, `app_state.py`, `errors.py` |

## 三、Go 版

| 版本 | 文件路径 | 端口 | 编译命令 |
|------|----------|------|----------|
| **api-proxy-go** | `api-proxy-go/main.go` | 5060 | `go build -o api-proxy-go.exe` |

**Go 版特性：**
- API Key 认证
- 请求限流
- 流量日志记录
- 故障自动转移
- 权重管理
- 资源占用低 (~10MB)

## 四、启动脚本对应关系

### 4.1 推荐使用的启动脚本（常用）

| 脚本 | 启动内容 | 端口 |
|------|----------|------|

| `036-start_all_services_gui.bat` | 启动 `start_all_services_gui.py` | 同上 |
| `035-start_all_services.bat` | 启动 free8 + 主服务(优化版) | 5000, 5008 |
| `050-start_api_proxy_go.bat` | 启动 Go 版 | 5060 |

## 六、快速启动指南

### 6.1 推荐方式（GUI）

```bash
# 方式1：双击启动 GUI
036-start_all_services_gui.bat

# 方式2：直接运行 Python
python start_all_services_gui.py
```

### 6.2 推荐方式（命令行）

```bash
# 启动所有服务（free5 + free8 + 主服务）
035-start_all_services.bat

# 或单独启动 Go 版
050-start_api_proxy_go.bat
```

### 6.3 独立启动

```bash
# 启动主服务（优化版）
cd multi_free_api_proxy
python multi_free_api_proxy_v3_optimized.py

# 启动独立服务
cd free_api_test/free8
python friendli_service.py
```
